# ERP – Cadena de Pizzerías
# Descripción del Proyecto
El presente proyecto consiste en el análisis y diseño de un sistema ERP (Enterprise Resource Planning) para una cadena de pizzerías, inspirado en el modelo operativo de empresas como Domino’s Pizza.

El sistema permitirá centralizar y automatizar los procesos operativos y administrativos, mejorando la gestión de inventarios, ventas, proveedores, recursos humanos y control financiero en múltiples sucursales.

# Objetivo General
Diseñar un sistema ERP que integre los principales procesos de una cadena de pizzerías, con el fin de optimizar la operación, reducir errores y facilitar la toma de decisiones mediante información centralizada y en tiempo real.

#Alcance del Sistema
El sistema ERP cubrirá los siguientes aspectos:
Gestión de inventarios (control de stock, vencimientos y alertas).
Registro de ventas en tienda física y pedidos en línea.
Gestión de proveedores y generación de órdenes de compra.
Control financiero (ingresos, gastos y reportes).
Gestión de recursos humanos (turnos y asistencia).
Módulo CRM (registro de clientes e historial de compras).
Soporte para múltiples sucursales.
Acceso por roles (administrador, cajero, gerente, entre otros).
